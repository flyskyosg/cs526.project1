Detailed Description

wandaid class This class creates a 3D model of a CAVE2 wand and labels it to give the user instructions on what the buttons do.

It's kind of like a tooltip for the wand. In CAVE2 when you raise your wand more than 1.5 meters wandaid will appear. Then disappear when you lower your wand. Note: To use this class you do not instantiate a class object. Instead you directly use the static member functions. To set the labels for the wand use the appropriate member function.
