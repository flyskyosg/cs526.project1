Introduction

Download and Install

  1. have omegalib and osgearth installed. You can find an instruction for Mac here
  2. download the source code from Github
  3. download data files at http://1234567, and put them directly in the project directory
  4. to run the project, do as following:
    5. >cd [omegalib_build]/bin/
    >./orun -s [where_you_put_the_project]/shiyin-p1.py

About the Data

Data used in this project is coming from different sources, including but not limiting to

  • City of Chicago Data Portal
  • Open Street Map
  • Yahoo! Map
  • OSGEARTH
